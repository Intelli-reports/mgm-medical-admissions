# React App

This is the active website project.

## Commands

```powershell
npm install
npm run dev
npm run build
npm run preview
```

## Main Folders

- `src/`
  - active React source
- `public/`
  - static assets
  - preserved legacy `.html` college pages used by homepage links
- `dist/`
  - production build output
- `archive/`
  - unused React-source experiments moved out of the active code path

## Active Pages

- `/`
- `/Blogs`
- `/blogs`
- `/contact`
- `/whatsappPage`

## Important Note

This app intentionally links some college/partner pages as standalone static HTML files from `public/`, because that matches the old uploaded build behavior.
